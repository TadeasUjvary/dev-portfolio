# ZEMKOP — demo web (zemní práce)

Ukázkový (demo) web pro obor **zemní a výkopové práce**. Slouží jako šablona pro prezentaci zájemcům.

Jedna stránka, `index.html` — světlá editorial-stavební varianta: pískové papírové pozadí, výrazná serifová typografie (Fraunces), stavební žlutý akcent, topografické vrstevnice a interaktivní řez půdními vrstvami. Záměrně odlišný design od ostatních demo webů (FERROX/kovovýroba je naopak tmavý).

## Stack

Statický web bez build kroku — Tailwind CSS (CDN), Alpine.js (CDN), Google Fonts (Fraunces / Hanken Grotesk). Nasaditelné kamkoliv jako statické soubory.

> Všechny firemní údaje (název ZEMKOP, kontakty, technika, reference) jsou smyšlené.
